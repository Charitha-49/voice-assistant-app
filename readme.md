#  AI Voice Assistant (Web App)

A simple voice-enabled AI assistant using Flask, JavaScript, and pyttsx3.

## Features

- Voice or text-based input
- Responds with text and voice
- Simple rule-based responses
- Background animation

## Tech Stack

- Flask (Python backend)
- HTML, CSS, JavaScript (frontend)
- pyttsx3 for text-to-speech

## Run Locally

```bash
git clone https://github.com/Charitha-49/voice-assistant-app.git
cd voice-assistant-app
pip install -r requirements.txt
python app.py
